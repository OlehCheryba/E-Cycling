If the hotkey isn't working in overlay

or fullscreen applications or games,

restart this tool as administrator.


To make this tool run on systems that use non-native

DPI-scaling: right click on the executable file, 'Properties', 

'Compatibility' tab, on Windows 10: enable 'Override high

DPI scaling behaviour' and set it to 'Application',

on Windows Vista, 7, 8: enable 'Disable display scaling

on high DPI settings'.
